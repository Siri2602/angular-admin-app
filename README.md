# Angular Admin App

A responsive Angular Admin Dashboard project with authentication, analytics, and dashboard management features.

## 🚀 Tech Stack

* Angular
* TypeScript
* HTML
* CSS

## 📦 Installation

```bash
npm install
```

## ▶️ Run Project

```bash
ng serve
```

Open:

```bash
http://localhost:4200
```

## 🏗️ Build Project

```bash
ng build
```
